Setup:

1. run `npm install`
2. create .gitignore
  * add .env
3. create .env file
  * add GUILD_ID
  * add CLIENT_ID
  * add TOKEN
  * add DB_HOST
  * add DB_DATABASE
  * add DB_USER
  * add DB_PORT
  * add DB_PASSWORD

Link to invite bot:
> https://discord.com/api/oauth2/authorize?client_id={CLIENT_ID}&guild_id={GUILD_ID}&scope=bot&permissions=8&applications.commands
